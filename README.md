# Cursor Meetup SLC

A community meetup for developers exploring AI-powered coding with Cursor in Salt Lake City.

## Event Details

- **Date:** March 13, 2025
- **Location:** Holladay, Utah
- **Host:** Mason Earl
- **Registration:** [Luma Event Page](https://luma.com/p521sri9)

## About

Join us for a Cursor SLC event, a community meetup for developers exploring AI-powered coding with Cursor!

Whether you're a Cursor beginner or power user, come connect with fellow developers, share tips and tricks, and learn how AI can transform your coding workflow.

## What to Expect

- Networking with local developers
- Cursor tips & tricks sharing
- Live coding demos
- Q&A session
- Coffee & snacks

All skill levels welcome!

This event is part of the [Cursor Community](https://cursor.com/community) - meetups, hackathons, and workshops taking place around the world.
