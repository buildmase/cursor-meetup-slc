# Banners & Branding - Cursor Meetup SLC

## Banner Requirements

### Types Needed
1. **Welcome/Entrance Banner** - Main event banner
   - Size: 3' x 6' or 4' x 8' (standard event banner)
   - Content: "Cursor Meetup SLC", date, location

2. **Registration Table Banner** - Smaller banner for check-in
   - Size: 2' x 4' or tabletop banner
   - Content: "Welcome - Check In Here" or similar

3. **Stage/Presentation Banner** (optional)
   - Size: 3' x 6'
   - Content: Cursor logo, event branding

## Design Elements
- Cursor logo (from cursor.com)
- Event name: "Cursor Meetup SLC"
- Date: "March 13, 2025"
- Location: "Holladay, Utah" (if space allows)
- Event tagline: "AI-Powered Coding Community"

## Print Shop Options

### Local Salt Lake City
1. **FedEx Office** - Multiple locations
   - Fast turnaround (24-48 hours)
   - Online ordering available
   - Standard banner sizes

2. **Staples** - Multiple locations
   - Competitive pricing
   - Banner printing services

3. **Local Print Shops**
   - May offer better pricing for bulk
   - Support local business
   - Custom sizes available

4. **Online Options**
   - Vistaprint
   - BannerBuzz
   - Allow 5-7 business days for shipping

## Pricing Estimate
- Standard banner (3' x 6'): $30-80
- Tabletop banner (2' x 4'): $20-50
- **Total estimate: $50-130** (depending on quantity and vendor)

- [ ] Create banner design mockup
